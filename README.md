# AutoTap

轻量级 Windows 桌面自动化操作录制与回放工具

录制 → 回放 → 循环 → 轮错

## 功能特性

- 🎬 **操作录制** — 录制鼠标点击（左键/右键）和键盘输入（字符、独立按键、组合快捷键），自动过滤系统快捷键
- ⏱️ **精准回放** — 按原始时间间隔回放，支持 0.5x \~ 4.0x 速度调节
- 🔁 **循环执行** — 支持有限次循环和无限循环
- 🔄 **轮错机制** — 为操作配置多个备选坐标/按键，循环回放时自动轮换，适应动态界面
- ⏸️ **暂停/恢复** — 回放过程中按 F10 暂停，再按 F10 从暂停位置继续，无需从头开始
- 📺 **回放悬浮窗** — 回放时屏幕右上角显示半透明进度悬浮窗，实时显示轮数进度（已完成/总轮数）
- ⌨️ **全局热键** — F9 录制 / F10 暂停与回放 / Esc 停止，基于 Win32 RegisterHotKey 实现，不与录制监听冲突
- 📋 **模板管理** — 保存操作序列为模板，随时加载复用
- 🖥️ **系统托盘** — 最小化到托盘后台运行，托盘菜单控制核心功能
- 📦 **双版本分发** — 绿色便携版（ZIP）和 Windows 安装包（Setup.exe）

## 使用指南

### 基本流程

1. 按 `F9` 或点击 **开始录制**，正常操作鼠标和键盘
2. 再按 `F9` 停止录制，操作序列显示在列表中
3. 调整回放速度和循环次数
4. 按 `F10` 或点击 **开始回放**，主窗口自动最小化，右上角显示回放进度悬浮窗
5. 回放中按 `F10` 暂停，再按 `F10` 从暂停位置继续
6. 按 `Esc` 停止回放

### 快捷键

| 快捷键   | 功能          |
| ----- | ----------- |
| `F9`  | 切换录制（开始/停止） |
| `F10` | 暂停/恢复回放，未回放时开始回放 |
| `Esc` | 停止录制或回放 |

### 回放悬浮窗

回放开始后，屏幕右上角自动显示半透明进度悬浮窗：

- **有限循环模式**：显示 `已完成轮数/总轮数`（如 `7/10`），附带进度条
- **无限循环模式**：显示已完成轮数（如 `15`）
- 悬浮窗不接受任何交互，点击穿透
- 回放完成或按 Esc 停止后自动关闭

### 操作列表管理

- **删除** — 右键操作 → 删除
- **编辑间隔** — 双击间隔单元格直接修改
- **插入轮错** — 右键操作 → 插入轮错，配置备选坐标或按键
- **保存/加载** — 保存为 JSON 文件或加载已有文件
- **保存为模板** — 将当前序列保存为模板供后续复用

### 轮错机制说明

轮错（Round-Robin Error Offset）允许为单个操作配置多个备选执行方案。回放时，第 1 轮执行原始操作，从第 2 轮起依次轮换执行备选操作，循环复用。

例如：某点击操作配置了 3 个轮错坐标 A、B、C，则：

- 第 1 轮 → 原始坐标
- 第 2 轮 → 坐标 A
- 第 3 轮 → 坐标 B
- 第 4 轮 → 坐标 C
- 第 5 轮 → 坐标 A（循环）

## 应用场景

### 电商运营

双十一零点抢券，手速永远慢半拍。用 AutoTap 提前录制一套抢券操作流程——打开活动页面、点击领券按钮、确认领取——设置无限循环，到点一键启动，比任何人的手都快。秒杀抢购也是同样的道理，录制从进入商品页到提交订单的完整路径，循环执行，直到抢到为止。

店铺每天要上架几十个商品，每个都要重复填写标题、价格、库存、上传图片、选择分类。录一次上架流程作为模板，之后每次只需要改改文字内容，让 AutoTap 替你点完那些永远一样的按钮和输入框。批量修改商品价格、统一调整运费模板，也是同理——录一遍，回放百遍。

客服每天回复大量重复问题，运费多少、什么时候发货、尺码怎么选。把常见回复录成模板，遇到对应问题直接加载回放，不用每次都手动打字。

### 游戏辅助

日常任务又臭又长，每天上线签到、领邮件、扫荡副本、领取奖励，一套流程点下来十几分钟。录制一次，以后每天启动 AutoTap 跑一遍就行，自己去倒杯咖啡回来全部搞定。

挂机刷材料时，很多游戏需要每隔一段时间点击确认、领取掉落、重新进入关卡。用无限循环回放，配合轮错机制应对按钮位置的微小偏移，稳定挂机不再是梦。

批量开礼包、一键分解低品质装备、连续强化装备到指定等级——这些机械操作交给 AutoTap，你只管享受游戏真正有趣的部分。

### 办公自动化

每个月做报表，要把数据从一个系统导出再粘贴到另一个系统，格式还要调整。录一遍从导出到粘贴到格式化的完整操作，每月一键搞定，再也不用对着屏幕重复点来点去。

HR 每月录入考勤数据，财务每月对账录入凭证，都是同样的界面、同样的操作步骤、不同的数据。用 AutoTap 录制操作流程，保存为模板，下次直接加载使用。

批量重命名文件、在 ERP 系统里逐条录入数据、在 CRM 里批量更新客户信息——任何需要你在固定界面重复点击和输入的场景，AutoTap 都能替你完成。

### 软件测试

功能测试中需要反复执行相同的操作路径来验证某个流程是否正常。录制标准操作流程，每次发版后回放一遍，立刻知道有没有回归问题。

做压力测试时，需要模拟用户连续操作数百次来观察系统表现。设置循环次数为 500，让 AutoTap 不知疲倦地重复操作，你只需要盯着监控面板看数据。

UI 回归测试中，界面按钮位置可能因为版本迭代发生微小偏移。轮错机制正好派上用场——为关键点击配置几个备选坐标，即使按钮位置略有变化也能准确命中。

### 数据采集

某些网站没有提供 API，需要手动翻页复制数据。录制翻页→选择→复制→粘贴的流程，循环执行，很快就能把几十页数据整理完毕。

批量下载文件时，每个下载链接都要点击确认、选择保存路径、等待完成。AutoTap 可以替你完成这些机械点击，你只需要确保网络畅通。

### 社交媒体运营

管理多个账号时，每个账号都要执行发帖、点赞、评论等操作。把每个账号的操作录成独立模板，切换账号后加载对应模板一键执行。

定时发布内容时，提前录制发布流程，到点启动回放即可。比手动操作更准时，也比任何定时插件更灵活——因为 AutoTap 模拟的是真实的鼠标键盘操作，不依赖任何平台接口。

## 从源码构建

### 打包为可执行文件

```bash
pip install pyinstaller
pyinstaller AutoTap.spec
```

输出在 `dist/AutoTap/` 目录。

### 构建安装包

需先安装 [Inno Setup 6](https://jrsoftware.org/isdl.php)，然后：

```bash
python build_installer.py
```

自动生成：

- `dist/AutoTap_x.x.x_Portable.zip` — 便携版
- `dist/AutoTap_x.x.x_Setup.exe` — 安装包
