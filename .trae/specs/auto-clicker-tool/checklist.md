# 自动化点击工具验收检查清单

本文档用于验证 AutoClicker 项目的完整性和功能正确性。每个检查项完成后应标记为已完成。

---

## 一、项目初始化验证

- [ ] 项目目录结构符合规范：src/、assets/、tests/、logs/、config/ 目录已创建
- [ ] requirements.txt 文件存在且包含所有必要依赖（PyQt6、pynput、pyautogui、pytest、pyinstaller）
- [ ] 虚拟环境可正常创建，依赖可正常安装
- [ ] Git 仓库已初始化（可选）
- [ ] .gitignore 文件已创建并包含正确的忽略规则

- [ ] OperationSequence 支持导入和导出 JSON 文件
